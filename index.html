    • Web, singkatan dari "World Wide Web," adalah sebuah sistem informasi global yang telah merevolusi cara manusia berinteraksi dengan informasi dan data melalui internet. Sejarah web dimulai pada akhir 1980-an ketika Tim Berners-Lee, seorang ilmuwan komputer di CERN, mengembangkan konsep web sebagai cara untuk berbagi informasi antarpeneliti. Pada tahun 1991, ia merilis protokol HTTP (Hypertext Transfer Protocol) dan browser pertama, yang dikenal sebagai "WorldWideWeb." Ini memungkinkan pengguna untuk mengakses dokumen yang terhubung dalam bentuk teks dan gambar dengan menggunakan tautan hiperteks.

    Cara kerja web lebih kompleks dan melibatkan beberapa komponen. Pertama, dokumen web disimpan di server web yang terhubung ke internet. Ketika seorang pengguna memasukkan URL (Uniform Resource Locator) ke dalam browser, browser mengirimkan permintaan HTTP ke server yang sesuai. Server tersebut menerima permintaan, mengidentifikasi dokumen yang diminta, dan mengirimkannya kembali ke browser pengguna dalam bentuk halaman web. Halaman web ini dapat berisi beragam media, termasuk teks, gambar, audio, dan video, serta tautan hiperteks yang menghubungkan ke halaman web lainnya.

    Selain itu, halaman web sering kali mengandung bahasa markup seperti HTML (Hypertext Markup Language) yang mengatur tampilan dan struktur halaman. CSS (Cascading Style Sheets) digunakan untuk mengendalikan tampilan visual, sedangkan JavaScript dapat memberikan interaktivitas pada halaman web. Framework web seperti Angular, React, dan Vue.js telah menjadi bagian integral dalam pengembangan web modern, membantu pengembang membangun aplikasi web yang lebih dinamis dan responsif. Informasi ini diintegrasikan oleh browser pengguna untuk menampilkan halaman web sesuai dengan tampilan yang kita lihat. Web juga mengandalkan konsep hyperlink, yang memungkinkan pengguna untuk berpindah dari satu halaman web ke halaman web lainnya dengan mengklik tautan. Ini membentuk jaringan informasi yang luar biasa luas yang memungkinkan pengguna menjelajahi dan mengakses berbagai sumber daya, mulai dari artikel ilmiah hingga media sosial, dari berbagai lokasi di seluruh dunia. Inilah dasar dari fenomena web yang telah mengubah cara kita belajar, berkomunikasi, dan berinteraksi dengan dunia secara keseluruhan.
